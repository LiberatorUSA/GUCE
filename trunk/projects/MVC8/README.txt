
================== OGRE

Note that Ogre is a manditory dependancy.

For MVC8 users it is recommended to download and install the SDK from www.Ogre3D.org
1) install the SDK
2) Start Visual Studio 2005 
3) Go to Tools->Options
4) Select Projects and "Solutions"->"VC++ Directories"
5) At the right top for "Show directories for:" select "Include files"
6) Add the following to the list: "$(OGRE_HOME)\include"
7) At the right top for "Show directories for:" select "Library files"
8) Add the following to the list: "$(OGRE_HOME)\lib"
