You can get the Premake tool at:
http://premake.sourceforge.net/download

-------  Microsoft Windows users:

Download the executable premake.exe and place it in the GUCEF\projects\premake directory if it is not already present.
You can use batch files to generate the project files and solution for your IDE

Note that VC6 is not supported and the source code will more then likely not compile using VC6, it is considered obsolete.
Visual Studio 2002 and 2003 may need to be extended with stl-port to get things working. Visual studio 2005 and newer should work
without any modifications.
